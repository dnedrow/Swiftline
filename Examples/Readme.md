To run examples

Install [cocoapods-rome](https://github.com/neonichu/Rome)

    $ gem install cocoapods-rome
Run `pod install`

---
To run any example:

    chmod a+x Example.swift
    ./Example.swift


For example to run `ColorizeExample.swift`

    chmod a+x ColorizeExample.swift
    ./ColorizeExample.swift
